**#12 출자금 대비 수익률 (IRR 기반)**
- 대시보드 또는 월말마감에 섹션 추가

### 계산 방식: 연간 XIRR (연초 리셋 가정)

> "작년말에 출자금 전액 회수하고 연초에 다시 전액 재투입했다고 가정"
> → 연단위로 깔끔하게 끊어서 계산

**현금흐름 시계열 (해당 연도만)**
```
날짜              금액               설명
─────────────────────────────────────────
연초(1/1)        -(연초 누적 출자잔액)   기초 투입 (음수)
금년 출자 건별    -(금액)               추가 출자 (날짜별, 음수)
금년 인출 건별    +(금액)               인출 (날짜별, 양수)
연말/오늘        +(연말잔액 + 금년순이익) terminal value (양수)
```

**terminal value** = 연말 누적 출자 잔액 + 금년 누적 순이익
- 연말 출자 잔액 = 연초 잔액 + 금년 출자 – 금년 인출
- 금년 순이익 = 영업 activity (credit – debit) 해당 연도 합계

**XIRR 풀기**
```
NPV(r) = Σ CF_t / (1 + r)^(d_t / 365) = 0
```
→ 연간 수익률 r (%) 직접 산출, 별도 환산 불필요

**구현 계획**
- 연초 출자 잔액: 개인 activity 전년도까지 누계
- 금년 출자/인출: journal_lines 개인 activity, 해당 연도, 날짜·금액 조회
- 금년 순이익: 영업 activity 해당 연도 (credit – debit)
- XIRR: 서버/클라이언트 어디서든 이분법 50회로 충분
- 프로젝트별 분리 표시 고려

**#13 대출 이자 계산 로직 수정 + 커스텀화**

### 현황 및 문제
현재 원리금균등상환 이자 계산이 일할계산 방식으로 추정되는데,
실제 은행 방식은 **전월 잔고 × 연이율 / 12** (월할계산).

```
현재(추정):  이자 = 잔고 × (연이율 / 365) × 해당월 일수
수정 후:     이자 = 전월말 잔고 × 연이율 / 12
```

원금 = 월 납부액 - 이자 (동일)

### 대출별 산식 커스텀
은행마다 방식이 다를 수 있음 → loans 테이블에 `interest_calc` 컬럼 추가 고려

| 옵션 | 설명 |
|------|------|
| `monthly` (기본) | 전월잔고 × 연이율 / 12 ← 은행 표준 |
| `daily_30` | 잔고 × 연이율 / 12 (30일 고정) |
| `daily_actual` | 잔고 × 연이율 / 365 × 실일수 |
