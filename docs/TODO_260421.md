## DB 설계

### `spending_plans` (지출예정 마스터)
```
id              uuid PK
name            text          -- "바닥공사 잔금", "서버 구독료" 등
type            text          -- 'recurring' | 'one_time'
amount          numeric
recurrence_day  int           -- recurring만. 매월 N일 (예: 25)
planned_date    date          -- one_time만. 지급예정일
account_id      uuid          -- 지출 계정과목 (FK → accounts)
counterparty_id uuid          -- 거래처 (FK → counterparties, nullable)
project_id      uuid          -- 프로젝트 (FK → projects, nullable)
note            text          -- 메모
status          text          -- 'active' | 'completed' | 'cancelled'
created_at      timestamptz
```

### `spending_executions` (집행 내역 - 대출/지출예정 통합)
```
id              uuid PK
source_type     text          -- 'loan' | 'plan'
source_id       uuid          -- loan.id 또는 spending_plan.id
planned_date    date          -- 지급예정일
amount          numeric
status          text          -- 'pending' | 'executed' | 'postponed' | 'cancelled'
note            text          -- 연기/취소 시 적요 작성
journal_id      uuid          -- 전표 연결 (FK → journals, nullable)
executed_at     timestamptz   -- 실제 집행일시
created_at      timestamptz
```

**대시보드는 `spending_executions`를 월별로 GROUP BY 해서 쿼리.**

```
pending
  ├─ 집행확인 → executed (전표 자동발행, executed_at 기록)

         ↙                      ↘
  loan_settlements          spending_executions
```

## 버그: 대출 이자 계산 수정 필요

**파일:** `src/app/(erp)/loans/[id]/page.tsx` → `calcSchedule()`

**문제:**
- 현재 `Math.round(interest)` → display용. 실제 balance 차감엔 float 사용 → 누적 오차
- 은행 프로그램과 금액 불일치

**수정 방향:**
```
이자 = TRUNC(전달기초원금 × 연이율 / 12)   ← Math.trunc, 소수점 버림
원금상환 = 월납부액(PMT, 정수) - 이자
잔액 = 전달잔액 - 원금상환
```

**현재 코드 문제점:**
```ts
// 현재 (잘못됨)
const interest = balance * r          // float 그대로
const repayment = monthlyPayment - interest
balance -= repayment                  // float 누적 오차

// rows에 Math.round는 display만, 내부 balance는 여전히 float
```

**수정 후:**
```ts
const pmt = Math.round(monthlyPayment)          // PMT 정수로 고정
const interest = Math.trunc(balance * r)         // TRUNC 소수점 버림
const repayment = pmt - interest
balance -= repayment
```
